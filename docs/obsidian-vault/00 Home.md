## Recommended Daily Loop

1. Create a new task note from [[Templates/Task Note]].
2. Define the outcome, acceptance criteria, and validation command.
3. Ask the agent to read that note and implement the task.
4. Let the agent update the code and write back the outcome.
5. Use backlinks and tags in Obsidian to keep context across sessions.
