# Vault Bootstrap

## Outcome

Create a minimal Obsidian vault inside the repository so project knowledge can live next to the codebase.

## Scope

- Add a dashboard note
- Add a project map
- Add a test map
- Add an agent and Obsidian workflow note
- Add a reusable task template

## Validation

- Confirm the files exist under `docs/obsidian-vault`
- Confirm the root `README.md` points to the vault

## Result

The vault was created under `docs/obsidian-vault` and linked from the repository README.
